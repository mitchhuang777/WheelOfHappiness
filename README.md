# WheelOfHappiness, 快樂轉盤

快樂轉盤用於新型態爐石加班台，每贈送訂閱10次會獲得轉盤機會1次，轉到的次數就表示這次要玩的遊戲場次

本程式碼使用100萬次來模擬測試轉盤的期望值
